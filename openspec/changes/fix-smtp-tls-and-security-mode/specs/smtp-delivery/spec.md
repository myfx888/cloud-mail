## 修改需求

### 需求:SMTP 发信必须保留用户安全模式配置
SMTP 发信流程必须保留用户提交的 `smtpSecure=0` 配置，禁止因默认值逻辑将其错误覆盖为 `1`。

#### 场景:保存并发信使用 `smtpSecure=0`
- **当** 用户保存 SMTP 账号且 `secure` 为 `0`
- **那么** 系统必须持久化 `0` 并在发信时按非 TLS 直连或显式 STARTTLS 规则执行，而不是隐式 TLS

### 需求:SMTP 发信附件字段必须匹配客户端约定
SMTP 发信流程必须按 `worker-mailer` 约定输出附件 MIME 字段。

#### 场景:发送带附件邮件
- **当** 邮件包含附件
- **那么** 系统必须输出 `attachments[].mimeType` 字段，且值与附件类型一致
